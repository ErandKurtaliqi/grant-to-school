#  Cleaning & Handling Missing Values

##  Objective
Prepare a clean and consistent dataset by managing missing, duplicated, and invalid data entries.

## ️ Steps Performed
1. Loaded the integrated dataset.
2. Filled missing district names with “Unknown District”.
3. Dropped rows with missing state names.
4. Removed duplicates.
5. Ensured scholarship amounts were valid (no negatives).

##  Output
- **dataset_cleaned.csv** – validated and clean dataset.
